# Nepeus Website

A premium one-page Next.js and Tailwind CSS landing page for Nepeus, an AI Agents & Business Automations company.

## Tech Stack

- Next.js
- React
- Tailwind CSS
- TypeScript

## Run Locally

Install dependencies:

```bash
pnpm install
```

Start the development server:

```bash
pnpm dev
```

Open `http://localhost:3000`.

## Production Build

```bash
pnpm build
pnpm start
```

## Deploy

The easiest deployment path is Vercel:

1. Push this folder to a Git repository.
2. Import the project in Vercel.
3. Keep the default Next.js settings.
4. Deploy.

## Notes

- Replace the placeholder WhatsApp link in `app/components/ContactForm.tsx`.
- Replace the `mailto:` form action with your preferred form service or backend endpoint before launch.
- The logo asset is stored in `public/nepeus-logo.png`.
